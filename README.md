This repository contains my **daily Excel practice files**. 
I am learning Excel from basics to intermediate level, covering formulas, logical functions, pivot tables, charts, and mini projects. 
All files are organized by **day of practice** to show consistent learning progress.


## Folder Structure

- **Day1_Basics/** : Basic formulas (SUM, AVERAGE, MAX, MIN), Sorting & Filtering  
- **Day2_Logical/** : Logical formulas (IF, Nested IF), Conditional Formatting  
- **Day3_Lookup/** : Lookup functions (VLOOKUP, XLOOKUP), Error Handling  
- **Day4_PivotTables/** : Pivot Tables for summarizing data  
- **Day5_Charts/** : Column, Bar, and Pie charts for data visualization  


## Skills Practiced

- Basic Excel formulas  
- Logical formulas (IF, Nested IF)  
- Lookup functions (VLOOKUP, XLOOKUP)  
- Pivot Tables  
- Charts & Data Visualization  
- Conditional Formatting  


## Usage

- Each folder contains Excel files for a specific day of practice.  
- Open each file to explore the formulas, formatting, pivot tables, and charts.  
- Designed for **self-learning or beginner Excel practice**.
